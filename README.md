# Libraries and Dependencies

- tailwind css
- timeago
- axios
- styled component
- react-router-dom
- node sass

# How to run application

- clone repository
- install dependecies  ` npm install `
- start application `npm start`
- open browser if not open and go to `http://localhost:3000`
